GPU-QUS GUI (Windows, GPU build)

Requirements:
- Windows 10/11
- NVIDIA GPU with NVIDIA driver installed

How to run:
1) Unzip the folder anywhere on a local drive.
2) Double-click:
   - launch_attenuation.bat
   - launch_bsc.bat

Notes:
- First run may take a few seconds.
- CUDA toolkit is NOT required.
- Outputs are saved to the folders selected in the UI.
